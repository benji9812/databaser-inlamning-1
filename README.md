# Musikbibliotek – SQL & Databasdesign

Detta projekt bygger ett komplett databasupplägg för ett musikbibliotek med tabellerna **Artist**, **Album** och **Track**.

## Tema
Musikbibliotek med fokus på relationer mellan artist, album och låtar.

## Filstruktur
- `sql/create_tables.sql` – tabellskapande med nycklar och constraints
- `sql/insert_data.sql` – seed-data för alla tabeller
- `sql/select_basic.sql` – grundläggande SELECT-frågor
- `sql/select_join.sql` – JOIN-frågor mellan tabeller
- `sql/updates.sql` – UPDATE-operationer
- `sql/deletes.sql` – DELETE-operationer
- `docs/er_diagram.md` – ER-diagram och datamodell
- `docs/rapport.md` – fullständig rapport
